###### **Overview**

HRMS (Human Resource Management System) is a comprehensive Human Resource Management System designed to support employee lifecycle management, leave requests, contracts, organizational structure, training, performance evaluation, asset management, electronic documents, fleet management, shift planning, and audit logging.



This project was developed as part of a Software Engineering graduation internship project.
